dotfiles
========
.bashrc, .vimrc, .gitconfg, ...etc


Support environment
========
* GNU/Linux
* Mac OSX


Local settings
===
* You can add local settings depends on your specific environment
 * `.zshrc.local`
 * `.gitconfig.local`


How to use
========
    cd ${HOME} && git clone https://github.com/ymkjp/dotfiles.git && sh dotfiles/setup.sh
